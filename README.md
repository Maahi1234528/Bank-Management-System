# Bank-Management-System
This Python-based application provides a comprehensive solution for managing banking operations through a secure, role-based interface. It features a dual-mode system for Staff and Customers, allowing for efficient record management and real-time financial transactions.

## Project Description
This is a Python-based Bank Management System designed to handle daily banking tasks digitally. It uses a "Staff Mode" for bank employees to manage records and a "Customer Mode" for users to manage their money.

The project stands out because it uses the Pandas library to show data in clean tables and includes a smart AI Advisor that gives financial tips based on your balance.

## Key Features
Two Login Modes: Separate menus for Bank Staff and Customers.

Tabular View: All data is displayed in professional grids using pandas.

Money Transactions: Easy Deposit and Withdrawal with real-time balance updates.

AI Financial Advisor: Automatically suggests if you should start an FD or reactivate a dormant account.

Smart Search: Find any customer instantly using their Name, ID, or Address.

Crash-Proof: Uses "Error Handling" so the program doesn't close if you type something wrong.


## Requirements
To run this project, you need:

Python 3.13 installed.

Pandas Library (Install it by typing pip install pandas in your terminal).


## How to Use
Run the script: python project_file.py

Choose Option 1 (Staff) to add new customers and create their accounts.

Choose Option 2 (Customer) to check your balance, deposit money, or get AI advice.

Data is stored in the system's memory during the session.



Customers Dictionary: Stores personal profiles.

Accounts Dictionary: Stores bank balances and account types.
